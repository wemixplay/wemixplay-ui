## Typography
<br /><br />

### Display 
``` css

 /* 
   
  각 타이포그래피에 대해 @mixin {fontname}-base 형식의 베이스 믹스인을 선언합니다.
  -base가 붙은 믹스인은 해당 폰트의 기본 font-size와 line-height, letter-spacing 값을 포함합니다.

  폰트의 굵기(weight)별로 개별 믹스인을 정의하며, 
  이때 해당 타이포의 베이스 믹스인을 @include한 후, 
  굵기(font-weight)와 오픈타입 글꼴속성 등을 별도로 지정합니다.

  
  Opentype 확인 : https://wakamaifondue.com/

*/


/* -base mixin */
@mixin display-01-base {
  /* 
    font-size: 56px, line-height: 72px인 타이포를 나타냄. 
    typo믹스인에서 rem으로 변환하므로 px 단위로 작성
  */
  @include typo(56, 72); 
  letter-spacing: -0.0319em;
}


/* font-weight, 오픈타입 속성을 지정한 세부적인 타이포 mixin */
@mixin display-01-b {
  font-weight: 700; /* weight */

  @include display-01-base; /* -base */
}

@mixin display-01-m {
  font-weight: 500; /* weight */

  @include display-01-base; /* -base */
  font-feature-settings: "tnum"; /* 오픈타입 속성 */
  font-variant-caps: all-small-caps; /* 오픈타입 속성 */
}


/* 선언된 타이포 mixin 사용 */
.typography { 
  @include display-01-m;
}

```
