## Spacing 
간격, 여백값에 적용되는 원칙입니다.  
단 2, 4, 6, 10, 12, 20, 28 네 숫자들만 작은 사이즈 컴포넌트들을 위해 예외적으로 허용되며, 20 이후 부터는 8 단위 로만 증가되어야 합니다.

``` css
$sp-2: 0.125rem;  // 2px
$sp-3: 0.188rem;  // 3px
$sp-4: 0.25rem;  // 4px
$sp-6: 0.375rem;  // 6px
$sp-8: 0.5rem;  // 8px
$sp-10: 0.625rem;  // 10px
$sp-12: 0.75rem; // 12px
$sp-16: 1rem; // 16px
$sp-18: 1.125rem; // 18px
$sp-20: 1.25rem;   // 20px
$sp-24: 1.5rem; // 24px
$sp-28: 1.75rem; // 28px
$sp-30: 1.875rem; // 30px
$sp-38: 2.375rem;   // 38px
$sp-32: 2rem; // 32px
$sp-34: 2.125rem; // 34px
$sp-40: 2.5rem;   // 40px
$sp-44: 2.75rem; // 44px
$sp-48: 3rem; // 48px
$sp-56: 3.5rem; // 56px
$sp-60: 3.75rem;   // 60px
$sp-72: 4.5rem; // 72px
$sp-80: 5rem;   // 80px
```
